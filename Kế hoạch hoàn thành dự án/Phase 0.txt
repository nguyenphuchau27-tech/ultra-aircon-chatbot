13. Tóm tắt ngắn gọn: file nào tạo mới, file nào nâng cấp
Tạo mới
project-brain.config.json
Nâng cấp
scripts/generate-project-brain.ts
File được generate tự động
PROJECT_BRAIN.md
Không cần sửa
package.json

vì script brain:generate của anh đã đúng rồi.

14. Cách kiểm tra nhanh sau khi làm xong

Chạy:

pnpm brain:generate

Nếu không lỗi, mở PROJECT_BRAIN.md và check 3 thứ:

Check 1

Có dòng phase hiện tại không:

- Current: PHASE 1 — BACKEND FOUNDATION
Check 2

Có roadmap không

Check 3

API total có khớp không:

- Total APIs: 30

API count 30 hiện tại đang khớp với bản anh đã generate trước đó.

15. Hướng tối ưu thêm

Sau khi bản này chạy ổn, bản nâng cấp tiếp theo nên là:

tách knownIssuesByPhase
tách doneItems
tự sinh Generated At
thêm decision log
thêm filesToTouchThisPhase

Nhưng hiện tại chưa cần làm ngay.
Bản phase-aware này là đủ chuẩn để bước vào Phase 1.

Bước tiếp theo đúng nhất bây giờ là: thay 2 file xong chạy pnpm brain:generate, rồi gửi cho em output mới để em kiểm tra trước khi mình vào audit backend foundation.
